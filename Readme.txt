# 📝 Task Manager - Java Swing

A desktop task manager application built with Java and Swing.  
This project allows users to manage tasks through a graphical interface, including adding, completing, removing, and persisting tasks using local file storage.

---

## 🚀 Features

- Add new tasks
- List all tasks
- Mark tasks as completed
- Remove tasks
- Persist tasks in a local `.txt` file
- Load saved tasks when the application starts
- Simple graphical interface using Swing

---

## 🛠️ Technologies Used

- Java
- Swing (GUI)
- File I/O (BufferedReader, FileWriter)
- Object-Oriented Programming (OOP)
